# Shizen Power – Site estático (logos da internet)

Pronto para GitHub + Netlify. As bandeiras de pagamento usam URLs do Wikimedia.
Se algum logo não carregar por política de referer, considere baixar a logo e colocar em `assets/`.

Publicação:
1. GitHub → Add file → Upload files → envie `index.html`, `assets/` e `netlify.toml` (não suba o .zip).
2. Netlify → Add new site → Import from Git → selecione o repositório → Publish dir `/`.
